Bits and Gits of EnCollab

All the information and docs that you will need to set up and run for the Encollab

Tools and Equipments

  1. Processor

    Preferably preferably (Core i5 3rd Generation or higher)

  2. Screens

    It is best to have a secondary screen to split your work in different view and also to check how your creation will look on different screen resolution. Especially if you are using a notebook to do the development work, the second monitor is strongly recommended. Alternatively, you can also check how the page will look on different screen via http://quirktools.com/screenfly/.

Communication

Team EnCollab uses the application called #Slack to communicate among developers. Different channels are created to discuss different topics or projects that we are working on. Skype is the preferred form of calling if you want to talk to another team member. Clear communication is imperative. Clear your doubt with your team members before you start working in order to not waste your time and effort.

  • ###Communication Tools
    • #Slack can be access in three ways
      1. Browser
      2. Desktop Apps
        • Mac
        • Windows
        • Linux
      3. Mobile Apps
        • iOS
        • Android
        • Windows Phone
    • __Good things about #Slack is:
    • when you are added to the Channel (i.e. Chat group), you can see the history of the conversation when the channel was first started. (thumb up)
    • When you enabled the push notification on your mobile device, it automatically disabled the e-mail notification. (very convenience logic)
