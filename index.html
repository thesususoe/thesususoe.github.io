Tools and Equipments

  1. Processor

Communication

Team EnCollab uses the application called #Slack to communicate among developers. Different channels are created to discuss different topics or projects that we are working on. Skype is the preferred form of calling if you want to talk to another team member. Clear communication is imperative. Clear your doubt with your team members before you start working in order to not waste your time and effort.

How to Set up the UI5 Development Environment

Step 1: Java Installation

Download the latest version of JDK from this link. After the download has finished, unzip the file into the directory where you want to install. Recommended location C:\Program Files\Java

After the JDK installation, set up the environment variable by following the steps below:

  • open command prompt, type "control sysdm.cpl"

  • go to Advanced tab, click "Environment Variables"

  • Click "New" for "User variables for Admin"
  • Variable name "JAVA_HOME", Variable value "Browse Directory" for the JDK installed location, and click OK.

  • Select "Path" on "System variables", click "Edit"

  • Click "New" on "Edit environment variable"
  • Enter "%JAVA_HOME%", and click "OK"

  • open command prompt, type "java -version"

If JDK is installed correctly, the following screen should display

Step 2: Web Server Setup

Download the latest and stable version of Apache Tomcat from this link. After the download has finished, unzip the file into the directory where you want to install. Recommended location C:\Program Files\ . After the installation, go to bin folder from command prompt and type "startup.bat"

You will see the following windows

Step 3: Eclipse Setup

Download the latest version of Eclipse IDE for Java EE developers from this link. After the download has finished, unzip the file into the directory where you want to install. Recommended location C:\Program Files\ . After the installation, click on eclipse.exe and create desktop shortcut. When the eclipse start for the first time, you will have to choose the specific location for your workspace. All your eclipse projects will store in that workspace.

Step 4: Add the web server to the Eclipse

Open Eclipse, go to "File > New > Others"

Choose "Server", click "Next"

Expand "Apache", choose the version of Tomcat that you installed.

After that click "Next", and "Finish"

After that, go to "Window > Show View > Servers"

You will see the new server has been added. Right click on the instance to start and stop the server.

  • if you receive the following error when starting the server.

  • you can either kill the process which is utilizing the port or change the port number of apache tomcat in the server.xml file located in the conf folder.

  • to kill the process, go to command prompt, enter "netstat -ano". Check the PID of the process which is using the desired port number. Then, enter taskkill /PID xxxx

Step 5: SAPUI5 installation

Open Eclipse, got to "Help > Install New Software"

Paste the following URL in the "Work with" field, and click "Add". You can get a list of updated URLs from the following link.

  • https://tools.hana.ondemand.com/mars/

Select the "UI Development Toolkit for HTML5" and click "Next".

You will see the overview of the features to be installed, click "Next".

Confirm the license agreement and click "Finish"

Once the installation is done, it will prompt to restart the Eclipse. Click "Yes".

> Simple Hello World UI5 web application

  • Open Eclipse, click "File > New > Others". Choose "SAPUI5 Application Development > Application Project"

  • Go to "index.html" and type Hello World in the body tag

  • Save the project. Run As > Run on Server > Choose the Tomcat, you will see the following screen

  • You can also Deploy on the Tomcat Server by exporting the project as .WAR

  • Right click on the project name. From the context menu, choose "Export > WAR file"

  • Copy the WAR file to the "webapps" folder in the Apache tomcat directory

  • Restart the apache by running "startup.bat" from the command prompt. You can access your web application under "http://localhost:8080/app_name/index.html"
